# 前言 #

<br/><br/>
&emsp;&emsp;本书结构和内容主要来源于 **《Lucene in Action Second Edition》** 英文原版，三位作者均来自 Apache Lucene 项目的核心成员：Michael McCandless，Erik Hatcher，Otis Gospodnetic´，出版于 2010 年，基于 **Lucene 3.0.1** 版本。此书论述的部分思想和原理目前依然有效，但经过十几年的迭代，Lucene 版本已更新至目前的 **8.11.2** 和 **9.3.0** 两个最新版本，虽然 Lucene 尽量保持版本的向后兼容性，但很大一部分理论和数据结构，及其背后的思想和算法都发生了翻天覆地的变化，致使很多用户级 API 也发生了巨大的变化，因此原书大部分内容已不适用于当前 Lucene 版本。本书依照原书的章节安排进行了重新编写，摈弃了当前版本不再支持的特性和用法说明，增加了一部分新增特性的论述，并尽量说明旧特性弃用的原因和移除版本，以及新特性引入的原因和版本。同时，书中尽量保留了原书的大部分图形和表格。对于测试代码，使用新版本 API 进行了重新编写和测试，也对相应的代码进行了增减，以对应本书的内容。

&emsp;&emsp;书中内容的另一部分来源于网络，包括一些图表插图，以增加对某些概念和细节的阐释，在此表示感谢和敬意\^_\^。

&emsp;&emsp;书中大部分细节内容来自于 Lucene 提供的官方文档和 JavaDoc。这是 Lucene 最权威最准确的阐释，是 Lucene 项目组多年的心血结晶。Apache 为 Lucene 项目提供了全面的支持，保留了 Lucene 所有版本的项目源代码和二进制 jar 包，以及在线文档，为我们提供了极大的便利。

&emsp;&emsp;本书内容基于 **Lucene 8.x**，示例代码使用当前最新 **8.11.2** 版本编译测试。由于 Lucene 主版本向后兼容特性，书中内容和代码示例可以在所有 Lucene 8.x 中运行。
